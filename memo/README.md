# Github 업로드 규칙



## 1. Commit

`날짜_폴더내용_파일내용_msg규칙_커밋내용`

ex) `210119_pythonBasic_pythonBasic_update_반복문내용추가`

| msg규칙 | 커밋내용                 |
| ------- | ------------------------ |
| upload  | 해당 파일을 처음 올릴 때 |
| delete  | 해당 파일을 제거할 때    |
| update  | 해당 파일을 수정할 때    |



## 2. 폴더 이름

`(순서_)내용`



## 3. 파일이름

`폴더이름_(순서_)파일내용`
